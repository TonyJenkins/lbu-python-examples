# The Great Random Game

A game slightly like Pontoon or Blackjack. But with a die.

In this game, the computer "thinks of" a number between 17 and 29, inclusive.

The player is shown numbers from the roll of a die, and must add 
these to get as close to the target as possible, above or below. After each 
number, the player must decide if they would want another.

To have a valid turn, player must end no more than 5 from the target.
